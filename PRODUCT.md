# Product

## Register

product

## Users

This is a personal learning project for a beginner practicing Codex, Git, GitHub, VS Code, HTML, CSS, JavaScript, and simple browser games.

## Product Purpose

The page records learning progress and provides small interactive exercises. Success means each new concept is visible, testable, and easy to understand.

## Brand Personality

Clear, practical, beginner-friendly.

## Anti-references

Avoid a commercial landing page, heavy decoration, complex layouts, and visual effects that hide the learning purpose.

## Design Principles

Keep each exercise easy to find.
Make controls look consistent.
Separate learning notes from game controls.
Prefer readable spacing over decoration.

## Accessibility & Inclusion

Use readable contrast, clear focus states, simple Chinese labels, and reduced-motion support for animations.
